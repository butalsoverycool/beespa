## Get started

1. Clone this repo
   ```bash
   git clone https://github.com/butalsoverycool/beespa.git
   ```
2. Enter the cloned repo and install the dependencies
   ```bash
   npm i
   ```
3. Start the dev server
   ```bash
   npm run dev
   ```
4. Preview site on localhost:3000
